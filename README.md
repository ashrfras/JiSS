# جيسس

## محاذاة العناصر

## الموضع

التحكم في موضع العنصر.

- **.موضع-ساكن**: العنصر يتبع تدفق المستند الافتراضي.
- **.موضع-تابت**: العنصر يظل ثابتًا في موضعه حتى عند تمرير الصفحة.
- **.موضع-مطلق**: موضع العنصر يكون نسبيا لأقرب عنصر ذو موضع نسبي.
- **.موضع-نسبي**: موضع العنصر يكون نسبيا لمكانه الأصلي في تدفق المستند.
- **.موضع-لصيق**: العنصر يصبح ثابتًا عند وصوله إلى موضع معين أثناء التمرير.
